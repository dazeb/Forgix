## ⚡ Why Forgix?

Forgix is an intelligent **Project Architect**. While other tools just clone repositories, Forgix remembers who you are. It automates the entire birth of a project—injecting personal details, fixing environment issues, and opening your workspace in seconds.

---

## ✨ Elite Features

* **🆔 Developer Profile:** Set your global identity once with `forgix config`. No more re-typing your name or license for every project.
* **🩺 Smart Doctor:** Built-in diagnostics that don't just find problems—they help fix them.
* **🎨 Visual Identity:** Professional ASCII art branding and categorized terminal menus.
* **📂 Smart Scaffolding:** Seamlessly handle local, remote (GitHub), or **Linked** custom templates from anywhere on your drive.
* **🧩 Plugin Injection:** Select multiple plugins (Docker, Tailwind, etc.) during project creation.

---

## 🛠️ CLI Commands

| Command | Description |
| :--- | :--- |
| `forgix config` | **(New)** Set your global developer name, license, and editor. |
| `forgix doctor` | Run health checks and auto-fix missing dependencies. |
| `forgix create` | Scaffold a personalized project in seconds. |
| `forgix link <name>`| Link any folder on your PC as a custom template. |

---

## 🚀 Speed Run Example

1. **Configure once:** `forgix config`
2. **Create forever:** `forgix create my-app --git --open`

Forgix will suggest your name, inject your license, initialize git, and open VS Code without you typing a single extra word.
