## ⚡ What is Forgix?

Forgix is an intelligent **Project Architect**. It automates the entire birth of a project—injecting personal details, initializing environments, and opening workspaces in seconds.

---

## ✨ Elite Features

* **🎨 Visual Identity:** Sleek ASCII art branding and categorized terminal menus.
* **🩺 Forgix Doctor:** Built-in diagnostics for your Node, Git, and VS Code setup.
* **👤 Personalization:** Interactive Author and License injection.
* **📂 Smart Scaffolding:** Choose from local, remote (GitHub), or **Linked** custom templates.
* **🧩 Plugin Injection:** Select multiple plugins (Docker, etc.) during project creation.

---

## 🚀 Installation

You can run Forgix instantly without installing it globally using `npx`:

```bash
npx @7h41c/forgix create
```

Or, install it globally to access the forgix command anywhere:

```bash
npm install -g @7h41c/forgix
```

---

## 🛠️ CLI Commands

| Command | Description |
|---------|-------------|
| `forgix doctor` | Runs a system diagnostic check. |
| `forgix list` | Displays all templates, plugins, and custom links. |
| `forgix create` | Starts the interactive scaffolding wizard. |
| `forgix link <name>` | Link any local folder as a Forgix template. |

## 🧩 Variable Injection
